Теги HTML с примерами

Тег Описание Пример использования
<head> Содержит мета-информацию о документе (ссылки на стили, шрифты, скрипты). <head><meta charset="UTF-8"><title>Мой сайт</title></head>
<h2> Заголовок второго уровня. <h2>Заголовок второго уровня</h2>
<h1> Заголовок первого уровня (основной заголовок). <h1>Основной заголовок страницы</h1>
<p> Создает абзац текста. <p>Это текст в абзаце.</p>
<i> Выделяет текст курсивом. <p>Текст с <i>курсивом</i>.</p>
<ul> Создает ненумерованный список. <ul><li>Первый пункт</li><li>Второй пункт</li></ul>
<li> Элемент списка. Может быть использован как в упорядоченных, так и в неупорядоченных списках. <ul><li>Элемент списка 1</li><li>Элемент списка 2</li></ul>
<em> Выделяет текст с акцентом (по умолчанию текст курсивом). <p>Это <em>важное</em> слово.</p>
<ol> Создает упорядоченный (нумерованный) список. <ol><li>Первый пункт</li><li>Второй пункт</li></ol>
<strong> Выделяет текст с сильным акцентом (обычно жирным шрифтом). <p><strong>Это очень важный текст!</strong></p>
<blockquote> Оформляет цитаты с отступом. <blockquote>"Это цитата, взятая из другого источника."</blockquote>
<div> Контейнер для группировки других элементов, часто используется для стилизации. <div class="container"><h2>Заголовок</h2><p>Текст внутри контейнера</p></div>
<pre> Отображает текст с сохранением форматирования (например, для кода). <pre>function hello() { console.log("Hello World"); }</pre>
