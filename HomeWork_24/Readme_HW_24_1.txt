1. Проблемы, решаемые с помощью фракций (fr) в CSS Grid:

Гибкое распределение пространства: Фракции позволяют автоматически распределять доступное пространство между колонками/строками.
Упрощение вычислений: Нет необходимости вручную вычислять размеры, всё настраивается через фракции.
Лучшее использование пространства: Элементы адаптируются под размер контейнера.
Пример:

css
Copy code
.grid {
  display: grid;
  grid-template-columns: 1fr 2fr; /* Первая колонка - 1 часть, вторая - 2 части */
}
2. Различия между медиа-запросами max-width и min-width:

max-width: Применяется для экранов шириной до указанного значения. Используется для мобильных устройств.

Пример:

@media (max-width: 600px) {
  .container {
    font-size: 14px;
  }
}
min-width: Применяется для экранов шириной от указанного значения. Используется для больших экранов.

Пример:

@media (min-width: 1024px) {
  .container {
    font-size: 18px;
  }
}

Использование:
max-width — для мобильных устройств.
min-width — для больших экранов.
